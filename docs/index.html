How to Install

1

Download

Click the download button for your operating system above.

2

Extract

Windows: Right-click the .zip file and select "Extract All".
macOS: Double-click the .zip file to extract it.

3

Open the App

Windows: Open the extracted folder and double-click "Fogo Validator Explorer.exe".
macOS: Drag "Fogo Validator Explorer.app" to your Applications folder and open it. If macOS says the app is from an unidentified developer, go to System Settings > Privacy & Security and click "Open Anyway".
